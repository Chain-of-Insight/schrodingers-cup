# API

## Development Environment

### Install dependencies

#### Install Realize

```
go get github.com/oxequa/realize
```

### Run local server

```
cd src/api
realize start
```

| Description | URI |
| ------------| --- |
| API Server | http://localhost:1323/ |
| Swagger Docs | http://localhost:1323/swagger/index.html |
| Realize Web UI | http://localhost:5002/ |
